# javascript-lotto-precourse : 로또

### 프로그래밍 요구사항

- 함수 길이는 15 라인 이내로 구현
- indent depth는 3을 넘지 않도록 구현
- 3항 연산자 사용 금지
- else 지양
- 함수가 한가지 일만 하도록 '단일책임원칙' 준수
- Jest 이용하여 단위 테스트 작성 (I/O 제외)

### 기능 정의서

구현 목록

1.  로또 구입 금액 입력 받기

    - 입력값은 자연수이고, 1,000 원 단위로 입력해야 함
    - 유효하지 않으면 `[ERROR]` 출력 후 재입력

2.  로또 발행

    - 로또 1장 가격은 1,000 원
    - 구입 금액에 따라 `구매 개수 = 금액 / 1,000` 만큼 로또 발행
    - 각 로또 번호는 오름차순 정렬

3.  당첨 번호 입력 받기

    - 숫자 6개 입력
    - 구분자: 콤마(`,`)
    - 1~45 범위의 정수
    - 중복 불가
    - 유효하지 않으면 `[ERROR]` 출력 후 재입력

4.  보너스 번호 입력 받기

    - 숫자 1개 입력
    - 당첨 번호와 중복 불가
    - 유효하지 않으면 `[ERROR]` 출력 후 재입력

5.  발행된 로또와 당첨 번호 비교

    - 당첨 내용 출력
    - 수익률 계산해 소수점 둘째 자리에서 반올림해 출력
    - `수익률 = 총 당첨 금액 / 로또 구매 금액\*100`
    - 예시) `100.0%`, `51.5%`, `1,000,000.0%`

      #### 로또 당첨 기준 및 당첨금액

      | 등수 | 기준                        | 당첨금액        |
      | ---- | --------------------------- | --------------- |
      | 1등  | 6개 번호 일치               | 2,000,000,000원 |
      | 2등  | 5개 번호 + 보너스 번호 일치 | 30,000,000원    |
      | 3등  | 5개 번호 일치               | 1,500,000원     |
      | 4등  | 4개 번호 일치               | 50,000원        |
      | 5등  | 3개 번호 일치               | 5,000원         |

6.  예외 처리

    - 모든 예외 메시지는 반드시 `[ERROR]`로 시작해야 함

    - 예외 발생 시:

      - 메시지를 출력한다.
      - 프로그램을 종료하지 않고 해당 단계부터 재입력

    - 프로그램 종료 시 process.exit()를 직접 호출하지 않는다.

7.  테스트

    - 테스트 항목

      1. 유닛 테스트
      2. 예외 처리 테스트

### 5번 목록의 출력 기준 세부적으로 구분하기

#### 오류 출력 기준

- 입력이 다음과 같을 경우에 오류 출력을 한다.

  - 경우 1.

    - 공백 입력
    - 처리: `[ERROR] 공백은 입력할 수 없습니다.`

  - 경우 2.

    - 로또 구입 금액이 `1,000` 단위가 아닐 경우
    - 처리: `[ERROR] 로또 구입 금액은 1,000 단위로 입력할 수 있습니다.`

  - 경우 3.

    - 콤마(`,`) 외 구분자 사용
    - 처리: `[ERROR] 숫자 구분은 콤마(,)로 구분해야 합니다.`

  - 경우 4.

    - 중복 숫자 입력
    - 입력(이름): `"9, 6, 22, 38, 38, 7, 9",`
    - 처리: `[ERROR] 중복된 숫자를 입력할 수 없습니다.`

  - 경우 5.

    - 숫자 길이 초과(보너스 숫자 포함 총 7개 초과)
    - 입력(이름): `"1,2,3,4,5,6,7,8"`
    - 처리: `[ERROR] 총 숫자 개수는 보너스 숫자 포함 7개입니다.`

  - 경우 6.

    - 번호 입력 받을 시 `1 ~ 45` 사이의 숫자가 아닌 경우
    - 입력(이름): `"46, ..."`
    - 처리: `[ERROR] 1부터 45 사이의 숫자만 입력할 수 있습니다.`
